Описание базы данных DNS_shop https://www.dns-shop.ru/.

База Данных создана для интернет-магазина электроники и бытовой техники DNS. База данных состоит из 12 таблиц.
Для каждой таблицы указываются: ее название, описание назначения. К таблицам используется связи
«1:много». Также для осуществления связи «много:много» используется таблица-связка.

Список таблиц:
catalog
city
discounts
manufacturers
orders
price
products
quantity
reviews
stores
user_adress
users


catalogs
Таблица для хранения названий разделов сайта (сортировка товара магазина по разделам). Связанна с таблицей products. 
Для курсовой работы выбраны 5 разделов: Микроволновые печи, Холодильники, Посудомоечные машины, Мультиварки, Плиты электрические.


manufacturers
Таблица для хранения данных о производителях товара магазина. Информация о производителях вынесена в отдельную таблицу, 
чтобы облегчить таблицу products, поскольку информация о производителях нужна не всегда, когда нужна информация о товаре. 
Связанна с таблицей products. Для курсовой работы выбраны 10 популярных производителей техники.

products
Таблица для хранения данных о товарах интернет-магазина. Таблица имеет связи с таблицами: catalogs, manufacturers, orders, price,
discounts, reviews, stores. Для курсовой работы произвольно выбраны по 2 товара для каждого из 5 разделов.

price
Таблица хранит данные о ценах на товар без учета скидок. Связанна с таблицей products. 

discounts
Таблица «Скидки». Здесь хранится информация о скидках на товар магазина. Связанна с таблицей products. 

city
Таблица сохраняет информацию о странах и городах. Связана с таблицами users (клиенты) и stores (магазины).

stores
Таблица хранит данные о физических магазинах и пунктах выдачи товара. Связана с таблицами quantity (таблица-связка по 
остаткам товара), orders (заказы), city (города).

users
Таблица для хранения данных о клиентах магазина. Связана с таблицами user_adress (адреса клиентов), orders (заказы), 
reviews (отзывы о товаре).

user_adress
Таблица хранит адреса клиентов интернет-магазина. Адреса клиентов вынесены в отдельную таблицу, чтобы облегчить таблицу users, 
т.к. адреса клиентов нужны не всегда, когда нужны другие данные о клиентах. Связана с таблицами users (клиенты) и city (города).

orders
Таблица заказов интернет-магазина. Связанные таблицы: products (товары), stores (магазины), users (клиенты).

quantity
Таблица отображает количество товара, доступного для продажи. Это таблица-связка разделов products (товары) и stores (магазины). 
Связанные таблицы: products и stores.

reviews
Таблица содержит отзывы о товаре, оставленные клиентами магазина. Связана с таблицами products и users.
